# Platforms

The [rust-lang/mdBook](https://github.com/rust-lang/mdBook/releases/latest)
project provides the pre-build `mdbook` binary which is used by `rules_mdbook`
(see the `MODULES.bazel` file).

| Operating System | Architecture |
| :--------------- | :----------- |
| macOS            | x86_64       |
| Linux            | aarch64      |
| Linux            | x86_64       |
| Windows          | x86_64       |
